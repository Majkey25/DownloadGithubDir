GitHub API

Free · Client-side · No limits

Download any GitHub folder as a ZIP.

Paste a GitHub URL or owner/repo/path. We fetch only the files you need and pack them instantly in your browser.

Token is used only in your browser to raise the rate limit.

Idle

How it works

Client-side app that walks the GitHub Contents API, streams each file, and zips on the fly with JSZip.

Privacy

No servers. Everything runs in your browser; optional tokens never leave your device.

Tips

  • Use full URLs for private repos with a token that has `repo` scope.
  • Keep directory sizes reasonable to avoid API limits.
