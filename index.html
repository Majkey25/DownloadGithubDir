DownloadGithubDir - Client-side

Download any GitHub folder

Download any GitHub folder as a ZIP, entirely in your browser.

Token stays in your browser; use if you hit rate limits or for private repos.

Idle

How it works

Uses the GitHub Contents API, streams files in parallel, zips with JSZip, no backend.

Privacy first

Everything runs locally in your browser. Optional tokens never leave the page.

Tips

Use full URLs for private repos with a token that has repo scope. Keep folders reasonable.
